# 叮咚特色 调用规则

> Figma node: `22716:25478`（前景图 bar）/ `22716:25520`（心智 tab）

## 用这个组件的条件
首页"叮咚特色品质之爱"模块，固定结构的品牌心智展示区，含活动入口和瓷片列表。

## 不用这个组件的条件
- 通用运营位 → 用 CMS 图片组件
- 商品宫格展示 → 用 `金刚位`
- 此组件结构固定，不适合改作其他用途

## 变体选择

无变体，结构固定。`瓷片列表` 传商品瓷片数据，`活动文字` 控制右侧活动入口文案。

## 示例

```tsx
<叮咚特色
  活动文字="天天赢免单"
  瓷片列表={[
    { label: '有机蔬菜', img: '/tiles/organic.jpg' },
    { label: '黑猪鲜肉', img: '/tiles/pork.jpg' },
    { label: '深海鱼虾', img: '/tiles/seafood.jpg' },
  ]}
  onActivityClick={handleActivity}
  onTileClick={(label) => navigate(`/category/${label}`)}
/>
```

## 常见误用

- 拆散瓷片列表单独复用（此组件是整块品牌模块，不支持拆散）
- 用此组件展示非品质之爱内容（背景/文字风格专属品牌场景）

## 可替代组件
