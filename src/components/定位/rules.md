# 定位 调用规则

> Figma node: `22716:7743`

## 用这个组件的条件
首页顶部地址栏，显示收货地址 + 站点名称，含消息入口。

## 不用这个组件的条件
- 地址列表/编辑页 → 不用此组件，用 `Cell` + `标题栏`
- 此组件仅用于首页顶部固定位置

## 变体选择

无变体，结构固定。

## 示例

```tsx
<定位
  地址="上海市浦东新区世纪大道XX号"
  站点="陆家嘴站"
  onAddressClick={() => navigate('/address')}
  onMessageClick={() => navigate('/message')}
/>
```

## 常见误用

- 在非首页顶部使用（此组件文字为白色，设计上叠于绿色背景）
- 放在绿色背景之外（文字不可见）
- 用定位组件展示非地址信息

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 二级页面顶部导航 | `标题栏` |
| 地址列表行 | `Cell`（GroupedRow） |
