# 价格文字标签 调用规则

> Figma node: `10145:67460`

## 用这个组件的条件
"今日疯抢"场景中非标准疯抢价格的特殊价格标签（尝鲜价、补贴价等）。

## 不用这个组件的条件
- 标准疯抢价 → 用 `疯抢价标签`
- 普通商品价格 → 直接用文字样式，不用此组件
- 已在 `今日疯抢` 组件内使用时由内部自动渲染

## 变体选择

| 场景 | 风格 |
|------|------|
| 显示价格数字 + 可选划线原价 | `粉底价格`（#FFEFEF 底+红色价格数字） |
| 尝鲜价、特惠价等文字描述 | `粉底文字`（#FFEFEF 底+红色文字） |
| 补贴价等强调场景 | `红底文字`（#FF3133 底+白色文字） |

## 关键 Props

- `风格="粉底价格"` 时传 `价格`（必填）和 `原价`（可选划线）
- `风格="粉底文字"` 或 `"红底文字"` 时传 `文字`

## 示例

```tsx
<价格文字标签 风格="粉底价格" 价格="6.5" 原价="23.88" />
<价格文字标签 风格="粉底文字" 文字="尝鲜价¥16.55" />
<价格文字标签 风格="红底文字" 文字="补贴价¥99.9" />
```

## 常见误用

- 用此组件展示标准疯抢价（标准疯抢价应用 `疯抢价标签`）
- 在今日疯抢之外大量使用（视觉风格强烈，属于场景专属组件）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 标准疯抢价格 | `疯抢价标签` |
| 普通促销文字 | `标签`（红色） |
