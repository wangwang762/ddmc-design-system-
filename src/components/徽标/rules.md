# 徽标 调用规则

> Figma node: `3787:17741`

## 用这个组件的条件
叠加在 icon、头像、商品图片角落的提示角标，绝对定位于父元素上。

## 不用这个组件的条件
- 独立展示的状态标签 → 用 `标签`
- 商品图片角落的文字状态（如"上新啦"）→ 用 `左下直角` 类型

## 变体选择

**类型（核心决策）：**

| 问题 | 类型 |
|------|------|
| 有没有新消息，不需要知道数量 | `红点` |
| 需要知道具体有几条/几件 | `数字` |
| 数量超过 99 | `更多`（自动展示 ···） |
| 贴在卡片/图片左上角的文字状态 | `左下直角` |

**尺寸：**

| 搭配的 icon | 尺寸 |
|------------|------|
| 24px 及以上（导航栏、头像） | `大`（16px） |
| 18px 及以下 | `小`（12px） |

**描边：**

| 背景 | 描边 |
|------|------|
| 白色/浅色背景 | `false`（默认） |
| 深色/彩色图片背景 | `true`（1px 白色边框） |

## 示例

```tsx
// 消息入口红点
<div style={{ position: 'relative' }}>
  <Icon name="Bell" size={24} />
  <徽标 类型="红点" 尺寸="大" style={{ position: 'absolute', top: 0, right: 0 }} />
</div>

// 购物车数量
<div style={{ position: 'relative' }}>
  <Icon name="Cart" size={24} />
  <徽标 类型="数字" 尺寸="大" count={cartCount} style={{ position: 'absolute', top: -4, right: -4 }} />
</div>

// 商品图左上角
<div style={{ position: 'relative' }}>
  <img src={productImage} />
  <徽标 类型="左下直角" 尺寸="大" label="上新啦" style={{ position: 'absolute', top: 0, left: 0 }} />
</div>
```

## 常见误用

- 用徽标展示商品属性文字（徽标只适合短数字/点/3字以内文字）
- 不用徽标而手写 `position: absolute` 红点，丢失设计一致性
- `左下直角` 类型错误地用在 icon 上（此类型专为图片左上角设计）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 商品属性行内标注 | `标签` |
| 图片上文字角标 | `商品图标签-单` |
