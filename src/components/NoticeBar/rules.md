# NoticeBar 调用规则

> Figma node: `8284:16421`

## 用这个组件的条件
页面顶部或内容区的通知条，传递系统消息、活动提醒、会员续费等信息。

## 不用这个组件的条件
- 需要用户确认的操作 → 用 `弹窗`
- 轻量自动消失的反馈 → 用 `Toast`
- 列表行信息 → 用 `Cell`

## 变体选择

| 场景 | 类型 |
|------|------|
| 系统通知、订单提醒等普通消息 | `一般`（深灰底，白字） |
| 需要用户注意的重要提示 | `重要`（黄底，橙字） |
| 绿卡到期续费提醒 | `绿卡`（浅灰底，绿卡logo） |
| 红包/活动续费提醒 | `红包`（浅灰底，红包图） |
| 限时活动带倒计时 | `倒计时`（深灰底，橙色倒计时） |

## 关键 Props

- `文案`：最多14个汉字，超出自动截断
- `iconName`：使用 Icon 组件名，适用于 `一般`/`重要` 类型的带图标样式
- `iconSrc`：图片 URL，适用于品牌图（绿卡logo、红包图、自定义图）
- `倒计时`：仅 `类型=倒计时` 时传，格式如 `"00:30:00.0"`
- `按钮文案` + `onButtonClick`：有 CTA 操作时传，无操作不传

## 示例

```tsx
// 一般通知（默认）
<NoticeBar 文案="12:06前您可在"我的订单"中追加商品" onClose={handleClose} />

// 重要通知带图标
<NoticeBar 类型="重要" 文案="请在截止时间前完成支付" iconName="Notice" 可关闭={false} />

// 绿卡续费
<NoticeBar
  类型="绿卡"
  文案="剩28天到期，续费平均省1728元/年"
  iconSrc="/icons/greencard-logo.png"
  按钮文案="去续费"
  onButtonClick={handleRenew}
  onClose={handleClose}
/>

// 倒计时活动
<NoticeBar
  类型="倒计时"
  文案="下单立减xx元！"
  iconSrc="/icons/hongbao.png"
  倒计时="00:30:00.0"
  按钮文案="去下单"
  onButtonClick={handleOrder}
  onClose={handleClose}
/>
```

## 常见误用

- 文案超过 14 个汉字不做截断处理（组件不自动换行）
- 用 NoticeBar 展示需要用户立即确认的重要操作（应用弹窗）
- 把 NoticeBar 放在页面中部而非顶部（此组件仅适合顶部通栏）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 需要用户确认的提示 | `弹窗` |
| 轻量自动消失反馈 | `Toast` |
| 页面内单行提示条 | `Notification` |
