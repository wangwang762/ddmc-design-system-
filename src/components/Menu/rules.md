# Menu 使用规则

Figma node: 10479:16891

## 场景

标题栏右上角"更多"按钮触发的弹出操作菜单，深色半透明背景，顶部有三角箭头指向触发器。

## 用法

```tsx
<Menu
  items={[
    { iconName: 'Share2',    label: '分享' },
    { iconName: 'widget',    label: '添加到小组件' },
    { iconName: 'Info circle', label: '规则' },
  ]}
/>
```

## arrowRight

`arrowRight` 控制箭头距容器右边的像素偏移（默认 18px），
用来对齐页面上的触发按钮位置。

## 注意

- 菜单宽度自适应，最大 315px
- 图标固定 16px，文字 14px 白色单行
- 通常配合绝对定位使用，叠在页面内容之上
- 遮罩/关闭逻辑由调用方实现（在 portal 或父组件中监听背景点击）

## 常见误用

- 把 Menu 用在非标题栏右上角位置（三角箭头样式专为从右上角弹出设计）
- 用弹窗替代 Menu 展示操作列表（弹窗有遮罩，Menu 无全屏遮罩，体验差异大）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 需要全屏遮罩的操作确认 | `弹窗` |
