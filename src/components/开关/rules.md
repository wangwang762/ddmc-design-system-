# 开关 调用规则

> Figma node: `3815:17862`

## 用这个组件的条件
设置项的开/关二态切换（通知、免打扰、权限等）。

## 不用这个组件的条件
- 互斥单选 → 用 `选择按钮`
- 多选 → 用普通 checkbox

## 变体选择

| 场景 | 尺寸 |
|------|------|
| 设置页主要开关项 | `大`（56×32px） |
| 列表行内开关 | `中`（48×24px） |
| 紧凑列表、卡片内嵌 | `小`（32×16px） |

## 示例

```tsx
const [enabled, setEnabled] = useState(false)

// 列表行内
<开关 尺寸="小" active={enabled} onChange={setEnabled} />

// 设置页
<开关 尺寸="大" active={notifyEnabled} onChange={setNotifyEnabled} />
```

## 常见误用

- 用开关表示"选中/未选中"（选择按钮才是正确语义）
- 尺寸选错：列表行用大开关，视觉过重

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 互斥单选 | `选择按钮` |
| 多选 | 原生 checkbox |
