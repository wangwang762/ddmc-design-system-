# 状态栏 调用规则

> Figma node: 状态栏

## 用这个组件的条件
页面顶部 iOS 状态栏（时间 + 电量/信号图标），用于 mockup 还原真实手机页面效果。

## 不用这个组件的条件
- 生产环境页面：状态栏由系统提供，不需要此组件
- 仅在设计稿还原/Storybook 演示时使用

## 变体选择

| 背景色 | theme |
|--------|-------|
| 深色/彩色背景（首页顶部渐变） | `light`（白色图标，默认） |
| 白色/浅色背景 | `dark`（深色图标） |

## 示例

```tsx
// 深色背景页面（首页）
<状态栏 theme="light" time="9:41" />

// 白色背景页面（详情页）
<状态栏 theme="dark" time="9:41" />
```

## 常见误用

- 在真实 App 代码中渲染状态栏（系统级组件，App 内由 iOS/Android 系统自动渲染）
- 用于非 mockup 的功能性页面（此组件仅用于设计稿截图展示）

## 可替代组件

无。状态栏仅用于 mockup 展示，无业务替代场景。
