# 选项卡 调用规则

> Figma node: 选项卡

## 用这个组件的条件
页面内容区域的 Tab 切换（商品详情的"详情/参数/评价"，分类页的分类切换）。

## 不用这个组件的条件
- 带排序功能的筛选条 → 用 `分段筛选器`
- 底部导航 → 用 `底部导航`

## 变体选择

**方向：**

| 场景 | 方向 |
|------|------|
| 页面顶部/内容区横向切换 | `横向`（默认） |
| 左侧固定分类导航（如分类页左栏） | `竖向` |

**横向列数（方向=横向时）：**

| Tab 数量 | 列数 |
|---------|------|
| 2 个 | `二列` |
| 3 个 | `三列` |
| 4 个 | `四列` |
| 5 个 | `五列` |
| 6 个及以上 | `滚动` |

## 示例

```tsx
// 商品详情页 Tab
<选项卡
  方向="横向"
  列数="三列"
  tabs={[
    { key: 'detail', label: '详情' },
    { key: 'params', label: '参数' },
    { key: 'reviews', label: '评价' },
  ]}
  activeKey={activeKey}
  onChange={setActiveKey}
/>

// 分类页左侧竖向导航
<选项卡
  方向="竖向"
  tabs={categoryList}
  activeKey={activeCategory}
  onChange={setActiveCategory}
/>
```

## 常见误用

- Tab 有排序功能时用选项卡（排序语义应用 `分段筛选器`）
- Tab 超过 5 个不切换为 `滚动` 列数，导致文字挤压
- 用选项卡替代底部 Tab 导航（底部导航有专属组件）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 带筛选/排序的条 | `分段筛选器` |
| 一级页面底部导航 | `底部导航` |
