# 商品卡片 调用规则

> Figma node: 单列 `4079:54295` / 双列 `4079:54736` / 三列 `4079:55465`

## 用这个组件的条件
需要展示可购买商品（有图片、价格、加购操作）时使用。

## 不用这个组件的条件
- 购物车列表、订单列表 → 改用 `Cell`
- 纯展示无加购操作 → 改用普通图文布局
- 菜谱内容卡片 → 不是商品，不用此组件

## 变体选择

| 页面 / 场景 | 布局 |
|------------|------|
| 搜索结果页 | `单列` |
| 新版分类页主列表 | `单列定高` |
| 新版分类页瀑布流 / 首页推荐双列区 | `双列定高` |
| 普通瀑布流 | `双列` |
| CMS 运营位 / 商详推荐 / 空间紧凑 | `三列` |
| 横向滚动泳道（如"今日疯抢"） | `泳道` |
| 新人专区 | `新人专区商卡` |

## 关键 Props

- `推荐理由`：仅 `单列定高` 可用，其他布局传了不渲染
- `榜单标签`：仅 `单列定高` 可用
- `榜单横幅`：`单列` / `双列` 可用，`三列` 空间不够禁止传
- `原价`：仅 `泳道` / `新人专区商卡` 可用，用于展示划线价
- `营销标签`：最多传 2 个，超出自动截断

## 示例

```tsx
// 搜索结果页
<商品卡片 布局="单列" 图片={url} 标题={name} 价格="38.76" onCartClick={handleAdd} />

// 首页双列推荐
<商品卡片 布局="双列定高" 图片={url} 标题={name} 价格="38.76" 下单数="2.1万人已下单" onCartClick={handleAdd} />

// 横向泳道
<商品卡片 布局="泳道" 图片={url} 标题={name} 价格="38.76" 原价="45.00" onCartClick={handleAdd} />
```

## 常见误用

- 购物车列表、订单列表用商品卡片（应用 `Cell`，购物车列表无加购按钮）
- 布局选错：三列位置放单列卡片，图片比例失调
- `榜单横幅` 传给三列卡片（三列空间不够，禁止传）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 购物车/订单列表行 | `Cell`（GroupedRow） |
| 纯内容展示无加购 | 自定义图文布局 |
