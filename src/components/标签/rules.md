# 标签 调用规则

> Figma node: `3617:16873`

## 用这个组件的条件
行内文字标签，标注商品/菜谱的属性、状态、来源等信息。

## 不用这个组件的条件
- 叠加在图标/头像上的角标 → 用 `徽标`
- 商品标题前的彩色业务标签（疯抢/预售等）→ 用 `标题前标签`
- 商品图片角落标签 → 用 `商品图标签-单`

## 变体选择

**类型：**

| 视觉需求 | 类型 |
|---------|------|
| 轻量标注（产地、品类、特性） | `基本`（浅底+彩色文字） |
| 强调标注（促销、新品、限时） | `实心`（主色填充+白字） |
| 次要强调，不遮挡背景 | `空心(实线)` |
| 非正式/预告性质 | `空心(虚线)` |

**颜色：**

| 语义 | 颜色 |
|------|------|
| 积极/正常（新鲜、自营、有货） | `绿色` |
| 紧迫/促销（促销、限时、缺货） | `红色` |
| 信息/特殊（预售、冷冻、进口） | `蓝色` |
| 普通属性 | `标准` |

**尺寸：**

| 场景 | 尺寸 |
|------|------|
| 详情页大标签、强调展示 | `28` |
| 卡片标准标签 | `24`（默认） |
| 紧凑卡片、标题行内 | `18` |
| 极小空间、副文本行内 | `14` |

## 示例

```tsx
<标签 类型="基本" 颜色="绿色" 尺寸="24">自营</标签>
<标签 类型="实心" 颜色="红色" 尺寸="24">限时抢</标签>
```

## 常见误用

- 把 `标签` 用在标题前替代 `标题前标签`（视觉样式完全不同）
- 把标签叠加在商品图片上（应用 `商品图标签-单`）
- 颜色硬编码 hex 而不用 token

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 商品标题前业务标签 | `标题前标签` |
| 商品图片上角标 | `商品图标签-单` |
| 数字/红点角标 | `徽标` |
