# 商品图标签-单 调用规则

> Figma node: 商品图标签-单

## 用这个组件的条件
叠加在商品图片上的单个颜色底标签（角标、浮层文字）。

## 不用这个组件的条件
- 商品标题旁的行内标签 → 用 `标签`
- 商品标题前的业务标签（疯抢/预售）→ 用 `标题前标签`
- 图标/头像上的数字角标 → 用 `徽标`

## 变体选择

| 场景 | 类型 |
|------|------|
| 促销、抢购、紧迫 | `粉红底` |
| 特色、买过 | `橙色底` |
| 活动、奖励 | `黄色底` |
| 自营、新鲜、绿色 | `绿色底` |
| 次要状态标注（白底） | `白底红字` |
| 中性状态（下架、已售罄） | `白底灰字` |

## 示例

```tsx
<商品图标签单 类型="粉红底">限时抢</商品图标签单>
<商品图标签单 类型="绿色底">自营</商品图标签单>
<商品图标签单 类型="白底灰字">已售罄</商品图标签单>
```

## 常见误用

- 放在图片外部作为行内标签（此组件专为叠加在图片上设计，需配合 position: absolute）
- 用 `标签` 叠加在图片上（标签无图片叠加视觉）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 商品标题旁行内标注 | `标签` |
| 商品标题前业务标签 | `标题前标签` |
| 图标/头像数字角标 | `徽标` |
