# 疯抢价标签 调用规则

> Figma node: `10145:21485`

## 用这个组件的条件
"今日疯抢"场次中的专属价格标签，渐变红底样式，带"抢"角标。

## 不用这个组件的条件
- 普通商品价格展示 → 直接用文字，不用此组件
- 促销文字/补贴文字标签 → 用 `价格文字标签`
- 已在 `今日疯抢` 组件内使用时，由 `今日疯抢` 内部自动渲染，无需手动使用

## 变体选择

无变体，只传价格字符串。

## 示例

```tsx
<疯抢价标签 价格="5.8" />
<疯抢价标签 价格="19.88" />
```

## 常见误用

- 在非"今日疯抢"场景使用（视觉强烈，用错场景破坏页面层次）
- 手写红色渐变价格文字而不用此组件（丢失设计一致性）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 非标准疯抢价（尝鲜价/补贴价） | `价格文字标签` |
| 普通促销标注 | `标签`（红色实心） |
