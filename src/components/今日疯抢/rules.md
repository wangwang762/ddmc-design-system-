# 今日疯抢 调用规则

> Figma node: 今日疯抢

## 用这个组件的条件
首页"今日疯抢"完整模块，含标题、场次倒计时、翻牌入口、横向商品泳道。

## 不用这个组件的条件
- 只需要展示单个疯抢商品 → 用 `商品卡片`（布局=泳道）
- 只需要疯抢价格标签 → 用 `疯抢价标签`
- 此组件是整块模块，不拆散复用

## 变体选择

无布局变体，结构固定。通过 props 控制文案和数据。

## 关键 Props

- `价格标签类型`（每个商品）：
  - `疯抢`（默认）→ 渐变红 `疯抢价标签`
  - `促销文字` → 粉底文字 `价格文字标签`（需同时传 `价格文字`）
  - `补贴文字` → 红底文字 `价格文字标签`（需同时传 `价格文字`）

## 示例

```tsx
<今日疯抢
  场次="14点场"
  倒计时小时={2}
  倒计时分钟={30}
  倒计时秒={0}
  副标题="金秋甜柚 低至¥2.8/个"
  商品列表={[
    { img: '/p1.jpg', price: '5.8', originalPrice: '26.9', 价格标签类型: '疯抢' },
    { img: '/p2.jpg', price: '16.55', 价格标签类型: '促销文字', 价格文字: '尝鲜价¥16.55' },
    { img: '/p3.jpg', price: '99.9', 价格标签类型: '补贴文字', 价格文字: '补贴价¥99.9' },
  ]}
  onMoreClick={() => navigate('/flash-sale')}
  onFlipClick={handleFlip}
/>
```

## 常见误用

- 拆散组件单独复用倒计时部分（此组件是整块模块，不支持拆散使用）
- 在非首页的普通促销场景套用此模块外壳（专属首页疯抢频道）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 单品疯抢展示 | `商品卡片`（布局=泳道）+ `疯抢价标签` |
| 仅需价格标签 | `疯抢价标签` / `价格文字标签` |
