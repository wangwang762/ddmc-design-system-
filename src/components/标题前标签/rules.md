# 标题前标签 调用规则

> Figma node: 标题前标签

## 用这个组件的条件
商品标题前的彩色业务标签（疯抢、预售、新品等），颜色和文字业务锁定，不可分离。

## 不用这个组件的条件
- 商品图片上的角标 → 用 `商品图标签-单`
- 通用属性标注（产地、品类）→ 用 `标签`
- 自定义文字+颜色的标签 → 用 `标签`（此组件颜色和文字不可自定义）

## 变体选择

| 类型 | 颜色 | 文字 | 使用场景 |
|------|------|------|---------|
| `疯抢` | 红色 #FC5455 | 疯抢 | 今日疯抢场景 |
| `预售` | 绿色 #00B740 | 预售 | 预售商品 |
| `新品` | 渐变绿 | 新品 | 新上架商品 |
| `趋势` | 橙色 #FF8A00 | 趋势 | 趋势爆款 |
| `宝妈严选` | 粉色 #FE6999 | 宝妈严选 | 母婴类目 |
| `叮咚心选` | 金色 #C9A856 | 叮咚心选 | 平台精选 |
| `美麒麟` | 深红 #EA333E | 美麒麟 | 品牌联名 |

**尺寸：**

| 场景 | 尺寸 |
|------|------|
| 商品卡片（默认） | `小` |
| 商品详情页 | `大` |

## 示例

```tsx
// 商品卡片标题前
<标题前标签 类型="疯抢" />
<标题前标签 类型="新品" />

// 商品详情页
<标题前标签 类型="叮咚心选" 尺寸="大" />
```

## 常见误用

- 用标题前标签自定义文字（此组件文字和颜色均业务锁定，不支持自定义，应用 `标签`）
- 把 `标签` 组件放在标题前（尺寸和视觉样式与标题前标签不同）

## 可替代组件

| 场景 | 替代组件 |
|------|---------|
| 自定义文字颜色的行内标签 | `标签` |
